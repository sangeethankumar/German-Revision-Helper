Comparisons with als (Komparativ + als)

SNK

October 12, 2025

Explanation: To compare two things, use the comparative adjective (-er form) followed by als (than).

Formula:

  • Adjective(-er) + als + comparison target

Examples:

  • Er ist größer als ich. → He is taller than me.
  • Dieses Modell läuft schneller als das alte. → This model runs faster than the old one.
